O `ButtonGroup` permite agrupar botões e gerenciá-los. Isso garante que os botões tenham o espaçamento correto.

```
<ButtonGroup>
  <Button content="Default" />
  <Button content="Default" />
  <Button content="Default" />
  <Button content="Default" />
  <Button content="Default" />
  <Button content="Default" />
</ButtonGroup>
```
